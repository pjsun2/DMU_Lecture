## ❗ CONFIG = 환경설정
$git config 설정값
```
$git config --unset
    → 기존 환경 파일 삭제
$cd 저장소 폴더
    → 저장소 폴더로 이동
$git config user.name "사용자 이름"
    → 선택한 로컬 저장소 이름 설정
$git config user.email "이메일 주소"
    → 선택한 로컬 저장소 이메일 설정
$git config --global user.name "사용자 이름"
    → 모든 로컬 저장소 이름 설정
$git config --global user.email "이메일 주소"
    → 모든 로컬 저장소 이메일 설정
```
